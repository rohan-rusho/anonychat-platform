🔧 Setup Instructions

  1. Install Node.js from nodejs.org
  2. Open terminal in project folder
  3. Run: npm install
  4. Run: npm start
  5. Visit: http://localhost:3000

🔒 Privacy First - Local Storage Only

This platform prioritizes your privacy:

  • 🎤 Live Audio/Video - No recording or storage
  • 📱 Device Storage Only - All preferences stored locally
  • 🚫 No Data Collection - Server only facilitates connections
  • 🔄 Real-time Streaming - Direct peer-to-peer communication

This is a demo page. To run the full chat platform, you need to install Node.js and start the server. See the SETUP.md file for detailed instructions.
